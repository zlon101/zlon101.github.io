##### 服务器搭建SSR

搭建 shadowsocks(SS) 或 shadowsocksR(SSR)，二选一。SSR是SS的增强版。

###### 搭建SSR

```
yum -y install wget

# 下载SSR脚本
wget --no-check-certificate https://raw.githubusercontent.com/teddysun/shadowsocks_install/master/shadowsocksR.sh

chmod +x ssr.sh
bash ssr.sh  显示引导界面
```
